# Contribution Guide

以下の技術要素を使っています。

- [Vue 3.x](https://github.com/vuejs/vue-next)
- [TailwindCSS](https://tailwindcss.com/)
- Firebase

## ライブラリのインストール

```
$ yarn install
```

## Vite でローカル起動して動作確認する

[Vite](https://github.com/vitejs/vite) でビルドしています。

```
$ yarn dev
```
